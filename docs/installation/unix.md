## Requirements
- At least 2 cores and 4Go+ of RAM
- A working UNIX (Linux, MacOS, ...) with bash
- A working installation of Docker
- minikube and kubectl

## Preparation
1. Clone the repo

## Launching
1. Run `./setup.sh`
2. It may ask to reboot if your user wasn't in the group `docker` as it should.
