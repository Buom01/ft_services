## Requirements
- At least 2 cores and 4Go+ of RAM

## Preparation
1. Ensure you have set core to 2+
2. Bootup
3. Clone the repo

## Launching
1. Run `./setup.sh`
2. It may ask to reboot if your user wasn't in the group `docker` as it should.

## Accessing
$ `minikube service [servicename]`
